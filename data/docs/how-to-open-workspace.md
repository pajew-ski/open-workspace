# Open Workspace System-Dokumentation

Willkommen im **Open Workspace**. Dies ist dein zentraler Hub für AI-gestützte Produktivität, Wissensmanagement und Projektplanung.

## Erste Schritte

Der Workspace ist in spezialisierte Module unterteilt, die zusammen ein "Second Brain" bilden.

### Der Workflow: Von der Idee zur Umsetzung

1.  **Erfassen**: Nutze `/docs` für Ideen oder Daily Logs.
2.  **Strukturieren**: Forme aus losen Notizen strukturierte **[[Modul: Aufgaben & Projekte]]**.
3.  **Visualisieren**: Nutze den **[[Modul: Canvas Visualisierung]]**, um komplexe Zusammenhänge aufzuzeichnen.
4.  **Ausführen**: Arbeite deine Tasks im Kanban-Board ab.

## Architektur der Module

- **Dashboard**: Deine Kommandozentrale. Zeigt aktuelle Aktivitäten und Quick-Links.
- **Dokumente**: Markdown-basiertes Wiki mit semantischer Suche.
- **Agenten**: Konfiguriere AI-Assistenten, die Zugriff auf diese Daten haben.
- **Kalender**: Integriere deine Termine (iCal).

## Semantic Data Layer (JSON-LD)

Dieses System ist mehr als nur Text. Jedes Dokument wird im Hintergrund als strukturierte Daten (`JSON-LD`) bereitgestellt.
- Ein `HowTo` (wie dieses) wird von Suchmaschinen als Anleitung erkannt.
- Ein `TechArticle` wird als technisches Fachwissen klassifiziert.

Probier es aus: Klicke auf die Links zu den Modulen, um mehr über deren spezifische Funktionen zu lernen.
